其实有更好的办法来创建斜向条纹，linear-gradient(线性渐变) 和 radial-gradient(径向渐变) 还有一个循环式的加强版 

      代码：
      
        background: repeating-linear-gradient(45deg, #fb3, #58a 30px);

查看代码实际上重复的部分是一个纯渐进的部分，咱们把代码稍微做一下改动，让这个重复的模块是条纹 

      代码：
      
        background: repeating-linear-gradient(45deg, #fb3, #fb3 15px, #58a 0, #58a 30px)
        可不能将这个0 写遗漏，写成这样：
        background: repeating-linear-gradient(45deg, #fb3, #fb3 15px, #58a, #58a 30px)

如果某个色标的位置值比整个列表中在它之前的色标的位置都要笑，则该色标的位置值会被设置为它前面所有色标位置值得最大值

当然这个角度可以任意调整,咱们将角度改为70度
background: repeating-linear-gradient(70deg, #fb3, #fb3 15px, #58a, #58a 30px)
