我们不能直接对main元素使用Blur滤镜，这样的话，整个main元素的内容都是模糊的了。1，关键点是我们将滤镜blur的应用对象，使用到了main的伪元素上面了。2，必须要将为元素的背景与body（或者是其他以此类推的父元素的）背景重合。 3，为什么要加margin: -30px; 因为模糊效果在接近边缘效果处会逐渐消退，所以我们要使这个伪元素的范围扩大（可以将上面注释的伪元素样式注释的background恢复，并且去掉margin在查看效果）。4， 因为我们使用了margin:-30px，这就使得main的伪元素面积大于main元素，这时候我们在main元素上 增加 overflow: hidden 达到目前这个效果。
